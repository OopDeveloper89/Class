Class

A javascript library lets you create more readable classes.

Hello everybody,

maybe you are here because you get know my javascript library Class.

I want to show you some examples what the library can do:

1) Define a class. For example you could have a Pizza.js file. 

// Pizza.js
Class.define('Pizza', {
   name : 'Pizza',
   price : 10
});

// In another *.js file you could create an instance of Pizza.
var pizza = Class.create('Pizza');

2) Doing inheritance:

// SalamiPizza.js
Class.define('SalamiPizza', {
   extend : 'Pizza', // key "extend" tells you the super class.
   name : 'SalamiPizza'
});

3) Work with constructor.

// SalamiPizza.js
Class.define('SalamiPizza', {
   extend : 'Pizza', // key "extend" tells you the super class.
   name : null,
   init : function() { // init() will be automatically called by creating a SalamiPizza.
      this.name = 'SalamiPizza';
   }
});

4) In addition the constructor can take as many parameters you want.

// SalamiPizza.js
Class.define('SalamiPizza', {
   extend : 'Pizza', // key "extend" tells you the super class.
   name : null,
   init : function(name, price) { // init() will be automatically called by creating a SalamiPizza.
      this.name = name;   
      this.price = price;
   }
});

// Just add parameters for the constructor after class name:
var salamiPizza = Class.create('SalamiPizza', 'SalamiPizza', 20);

Well, that`s it. If you have any suggestions I would be happy for feedback.

Have a nice day and lots of fun with class.js. :-)
